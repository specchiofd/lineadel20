# La Linea del 20 Digitale

Una versione digitale interattiva della **Linea del 20** del metodo Bortolato per l'apprendimento della matematica.

## Descrizione

Questo progetto è un'implementazione web della Linea del 20, uno strumento didattico sviluppato da Camillo Bortolato per facilitare l'apprendimento del calcolo e dei numeri nei bambini della scuola primaria.

## Funzionalità

- **20 tasti interattivi**: divisi in rosso (1-10) e verde (11-20)
- **Animazione dei tasti**: i tasti si alzano e abbassano al click
- **Contatori in tempo reale**:
  - Numero di tasti alzati
  - Somma dei valori dei tasti alzati
- **Nascondi/Mostra numeri**: per esercitarsi senza vedere i valori
- **Azzera tutto**: reset rapido di tutti i tasti
- **Design responsive**: funziona su desktop e dispositivi mobile
- **Raggruppamento visivo**: spaziatura ogni 5 tasti per facilitare il conteggio

## Come utilizzare

1. Apri il file `lineadel20.html` in un browser web
2. Clicca sui tasti per alzarli o abbassarli
3. Osserva i contatori che mostrano quanti tasti sono alzati e la loro somma
4. Usa "Nascondi numeri" per esercitarti senza vedere i valori
5. Usa "Azzera tutto" per resettare tutti i tasti

## Requisiti

- Un browser web moderno (Chrome, Firefox, Safari, Edge)
- Nessuna installazione o dipendenza esterna richiesta

## Tecnologie

- HTML5
- CSS3 (con variabili CSS e animazioni)
- JavaScript vanilla (nessuna libreria esterna)

## Note

Il progetto include riferimenti a file audio opzionali (`click-up.mp3`, `reset.mp3`) che possono essere aggiunti per migliorare l'esperienza interattiva.
